# MP-LogManager v2.0.0 (GAS Edition) Release Notes

このリリースは、データのプライバシー保護を究極まで高めるための **「GAS Edition」** へのメジャーアップデートです。

## 🛡️ 重大な変更 (Breaking Changes)
- **データ保存先の変更**: 従来の GitHub レポジトリ内保存から、ユーザー個人の **Google Drive** への保存へ完全に移行しました。
- **通信方式の刷新**: GitHub API に依存せず、独自の **Google Apps Script (GAS)** API を介して通信を行います。これにより、パブリックリポジトリを利用していても、外部に個人データが漏洩するリスクをゼロにしました。

## 🚀 新機能 (New Features)
- **Google Apps Script (GAS) 連携**: 
  - 設定画面から独自の GAS ウェブアプリ URL を設定することで、即座に Google Drive 同期が開始されます。
- **キャッシュバスティング対応**: 
  - ブラウザや PWA の強力なキャッシュを回避し、常に最新のデータを取得するためのタイムスタンプ制御を導入しました。
- **ドキュメント全面刷新**: 
  - ユーザーマニュアル、開発者ガイド、APIリファレンス、仕様書をすべて GAS 連携仕様に書き換えました。

## 🛠 改善・修正 (Enhancements & Fixes)
- **クリーンリポジトリ**: Git 履歴に過去の機密データが含まれない、完全にクリーンなリポジトリとして再構築されました。
- **PWA の安定性向上**: `sw.js` の更新により、オフラインとオンラインの切り替えがよりスムーズになりました。

## ⚠️ デスクトップ版についての注意
- 現在、デスクトップ版 (`sync_json.py`) の GAS 同期機能は開発中（移行準備中）です。
- デスクトップ版で Web App とのデータを共有する場合は、次回のアップデートをお待ちください。

## ⚙️ セットアップ
1. README の「セットアップ手順」に従い、GAS をデプロイしてください。
2. Web App 上の設定(⚙️)から GAS URL を入力してください。
