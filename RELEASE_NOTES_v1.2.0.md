# Release Notes - v1.2.0 (2026-02-12)

## 📌 概要
iPhone（Web App）とのデータ同期を完全に自動化し、Windows環境特有の日本語文字化け問題を解消した安定版アップデートです。

## ✨ 新機能
- **📱 モバイル同期の自動化**: `Run_LogManager.bat` を起動・終了するだけで、iPhone側の変更を自動取得し、PC側の変更を自動送信するようになりました。
- **🛠️ 同期ツールの強化**: `sync_json.py` に `--export` / `--import` オプションを追加し、より柔軟なデータ管理が可能になりました。
- **📝 内部ロギングの導入**: 同期実行時の詳細を `_sync_internal_log.txt` に記録し、コマンドプロンプト上でのエラー追跡性を向上させました。

## 🔧 改善・修正
- **🔤 日本語文字化けの解消**: Gitのバイナリ変換抑止（`.gitattributes`）およびPythonスクリプト内の明示的なUTF-8処理により、iPhoneブラウザでの文字化けを完全に修正しました。
- **⚙️ バッチファイルの安定化**: 特殊文字によるバッチ実行エラーを回避するため、記述方法を改善しました。

## 📦 アップデート方法
最新のリポジトリをプルするか、`Run_LogManager.bat` を含む全ファイルを上書きしてください。
