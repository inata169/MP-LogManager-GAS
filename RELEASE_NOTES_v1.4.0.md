# MP-LogManager v1.4.0 Release Notes

このリリースには、Webアプリケーション（JournalおよびTasks）の実用性とUXを大幅に向上させる新機能や改善が含まれています。

## 🚀 新機能 (New Features)
- **Journal Rich Editor (EasyMDE導入)**
  - Web AppのJournal入力欄をリッチエディタ化しました。
  - プレビュータブや各種マークダウンの記号ボタン、コードのシンタックスハイライト、スムーズなスクロールが利用可能になりました。
  - PCでは左ナビゲーション・右エディタのスプリットビューに対応しました。
- **Journal Web印刷・PDF保存**
  - エディタ右上に印刷ボタンを追加しました。
  - `marked.js` による高速なバックグラウンドレンダリングを行い、余計なUIを除外した状態で出力されます。
  - iPhoneでも「自動印刷警告」によるエラーを出さずに、スムーズにAirPrintやPDF保存へ連携できます。
- **Journal オートセーブ（下書き保存）**
  - 入力内容を5秒ごとにブラウザ(localStorage)へ自動保存します。不意にページを閉じても次回アクセス時に復元することが可能です。
- **タスク並べ替え機能 (Task Sorting)**
  - タスク一覧を「デフォルト順」「期限が近い順」「期限が遠い順」「新着順」で並べ替える機能を追加しました。
  - 選択したソート順は保存され、次回起動時にも維持されます。
- **検索機能 (Search)**
  - 過去の全Journalエントリや、保存されているタスクを横断検索できるサーチバーをそれぞれ追加しました。

## 🛠 改善・修正 (Enhancements & Fixes)
- **Markdown / テーブル表示の適正化**:
  - `marked.js` に全面移行し、Windows由来の改行(`\r\n`)等によるテーブルの表示崩れを解決しました。
- **設定ボタンの追加**:
  - ヘッダー右上に設定(⚙️)ボタンを追加し、いつでもGitHubトークンの確認・修正ができるようになりました。
- **iPhone / モバイル最適化**:
  - 各種モーダルUI、エディタの高さ、ツールバーが画面内に収まるようにレスポンシブデザインの調整を行いました。

## ⚙️ How to Update
常にGitHub Pagesで稼働しているWebアプリのため、PWA版またはブラウザでアクセスし、画面上のリロードボタン（またはPWAの再起動）で最新バージョンが適用されます。ローカルPCアプリを使用している場合は自動的に同期が行われます。
