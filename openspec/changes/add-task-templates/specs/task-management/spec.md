## ADDED Requirements

### Requirement: Task Templates
The system SHALL provide a way for users to select from predefined task templates to quickly fill out the task creation form.

#### Scenario: User selects a template
- **WHEN** the user opens the "Add New Task" dialog
- **AND** selects "Daily Morning QA" from the template list
- **THEN** the Title, Category, and Details fields are automatically populated with the template's data
