# Release Notes - MP-LogManager v1.1.0

**リリース日**: 2026-01-29

## 🎉 メジャーアップデート

MP-LogManager v1.1.0 は、v1.0.0からの大幅なアップデートです。
ダークモード、月次レポート、印刷機能、包括的なドキュメントが追加されました。

---

## ✨ 主要機能

### 1. タスク管理
- ✅ タスクの作成・編集・削除
- ✅ ステータス管理（未完了/完了）
- ✅ カテゴリ分類（Treatment Planning, Planning, QA, Study, Other）
- ✅ 優先度設定（High, Medium, Low）
- ✅ 期限設定
- ✅ リアルタイム検索バー
- ✅ ソート・フィルタリング機能
- ✅ 複数選択削除（Ctrl/Shift対応）
- ✅ テンプレート機能（定型タスクの素早い作成）
- ✅ CSV エクスポート/インポート
- ✅ **PDF印刷機能** 🆕

### 2. ダッシュボード
- ✅ サマリーカード（総タスク数、完了数、未完了数）
- ✅ 週間生産性グラフ（matplotlib統合）
- ✅ カテゴリ別進捗バー
- ✅ **PDF印刷機能** 🆕

### 3. Journal（日記機能）
- ✅ 複数エントリ対応（1日に複数の日記を記録可能）
- ✅ 日付ナビゲーション（前日/翌日ボタン）
- ✅ タスク同期機能（完了タスクを日記に自動挿入）
- ✅ スニペット機能（定型文の素早い挿入）
- ✅ **PDF印刷機能** 🆕

### 4. 月次レポート 🆕
- ✅ 年月選択ドロップダウン
- ✅ 日別完了タスク数グラフ（matplotlib）
- ✅ カテゴリ別集計表
- ✅ **PDF印刷機能** 🆕

### 5. 通知機能
- ✅ デスクトップ通知（Reminders）
- ✅ 期限切れ間近のタスクアラート

### 6. データ管理
- ✅ CSV エクスポート/インポート
- ✅ 自動バックアップ（起動時）
- ✅ クラウド同期対応（`config.json`でDB パス設定可能）

### 7. セキュリティ
- ✅ PHI（個人健康情報）検出・警告
  - 患者ID（6-8桁の数字）検出
  - 生年月日パターン検出

### 8. UI/UX 🆕
- ✅ **ダークモード**（ライト/ダーク切り替え）
- ✅ 設定の永続化（config.json）
- ✅ レスポンシブレイアウト

### 9. 印刷機能 🆕
- ✅ 全メニュー（Tasks、Journal、Dashboard、Monthly Report）でPDF出力対応
- ✅ 日本語フォント完全対応（MS Gothic）
- ✅ ファイル保存先選択可能

### 10. Windows対応
- ✅ UTF-8対応起動バッチファイル（`Run_LogManager.bat`）
- ✅ 日本語完全対応

- **グラフ描画**: matplotlib

## 📦 インストール

### 必要な依存関係
```bash
pip install -r requirements.txt
```

### 起動方法
```bash
# Windows
Run_LogManager.bat

# または直接実行
python main.py
```

詳細は `docs/USER_MANUAL.md` を参照してください。

## 🎯 使用方法

1. **タスク管理**: 「Tasks」タブでタスクを追加・管理
2. **日記**: 「Journal」タブで日々の記録を作成
3. **統計確認**: 「Dashboard」タブで生産性を可視化
4. **月次レポート**: 「Monthly Report」タブで月間の実績を確認
5. **印刷**: 各タブの「🖨 Print」ボタンでPDF出力
6. **ダークモード**: メニューバー「View → Toggle Dark Mode」で切り替え
