# MP-LogManager ユーザーマニュアル
**Version:** 2.0.0  
**最終更新:** 2026-03-31

---

## 1. はじめに
MP-LogManagerは、医学物理士の日常業務をサポートするタスク管理・日報作成プラットフォームです。
v2.0.0より、**Google Apps Script (GAS)** と連携し、**Google Drive** をデータの保存先とする「GAS Edition」へ移行しました。これにより、GitHub Pages (PWA) の利便性を享受しつつ、自分専用の安全なデータ管理を実現しています。

### 主な特徴
- **📓 Journal (日報)**: EasyMDEによるMarkdownリッチ編集、オートセーブ、Web印刷機能。
- **✅ Tasks (タスク管理)**: カテゴリ管理、期限ソート、リアルタイム検索。
- **📱 PWA対応**: iPhone/Androidのホーム画面に追加してネイティブアプリのように利用可能。
- **🛡️ プライバシー重視**: 個人データはすべてあなたの Google Drive に保存されます。

---

## 2. Web App のセットアップ (必須)
Web Appはブラウザから専用の GAS Web App URL を介して自分自身の Google Drive と通信します。

### 2.1. アクセスとホーム画面への追加
1. ブラウザでWeb AppのURL（例: `https://<user>.github.io/MP-LogManager-GAS/web/`）を開きます。
2. **iPhone**: Safariの「共有」メニューから「ホーム画面に追加」を選択。
3. **Android**: Chromeのメニューから「ホーム画面に追加」を選択。

### 2.2. GAS連携設定 (初回必須)
画面右上の **設定(⚙️)** ボタンをクリックして以下を設定します。

1. **GAS Web App URL**:
   - Google Drive で公開（デプロイ）した Google Apps Script の「ウェブアプリ URL」を入力します。
   - `https://script.google.com/macros/s/.../exec` で始まる文字列です。
   - 保存ボタンを押すと、以降のデータ同期にこの URL が使用されます。

## 3. 機能解説

### 3.1. Journal (日報)
- **リッチエディタ**: Markdown形式で入力。ツールバーから太字、リスト、リンク、画像などを挿入可能。
- **オートセーブ**: 入力内容は5秒ごとにブラウザの `localStorage` に保存され、ページを閉じても復元されます。
- **PDF印刷**: 「🖨️ 印刷」ボタンをクリックすると、UIを除外した状態で綺麗にPDF出力されます。

### 3.2. Tasks (タスク管理)
- **新規タスク**: 「+ 新規」ボタンから追加。カテゴリ、優先度、期限を設定します。
- **検索・フィルタ**: 検索バーでタイトルや詳細を瞬時に絞り込めます。
- **ソート**: 期限が近い順や、作成日が新しい順などに並べ替え可能です。

---

## 4. トラブルシューティング
- **同期エラー**: 保存した GAS URL が正しいか確認してください。
- **読み込みが古い場合**: ヘッダーの更新ボタンを押すか、PWA を再起動してください。
- **文字化け**: 全てのデータファイル（JSON）は UTF-8 で保存されます。
