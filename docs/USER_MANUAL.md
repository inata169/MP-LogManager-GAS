# MP-LogManager ユーザーマニュアル
**Version:** 1.4.1  
**最終更新:** 2026-03-25

---

## 1. はじめに
MP-LogManagerは、医学物理士の日常業務をサポートするタスク管理・日報作成プラットフォームです。
v1.4.0以降、**Web App (PWA)** をメインインターフェースとし、モバイル・PCのあらゆる環境からGitHub経由でデータを直接同期・管理できるようになりました。

### 主な特徴
- **📓 Journal (日報)**: EasyMDEによるMarkdownリッチ編集、オートセーブ、Web印刷機能。
- **✅ Tasks (タスク管理)**: カテゴリ管理、期限ソート、リアルタイム検索。
- **📱 PWA対応**: iPhone/Androidのホーム画面に追加してネイティブアプリのように利用可能。
- **📊 汎用設計**: 自身のリポジトリをフォークして、簡単な設定で即座に運用開始。

---

## 2. Web App のセットアップ (推奨)
Web Appはブラウザからアクセスし、GitHub APIを介してデータを保存します。

### 2.1. アクセスとホーム画面への追加
1. ブラウザでWeb AppのURL（例: `https://<user>.github.io/MP-LogManager/web/`）を開きます。
2. **iPhone**: Safariの「共有」メニューから「ホーム画面に追加」を選択。
3. **Android**: Chromeのメニューから「ホーム画面に追加」を選択。

### 2.2. GitHub連携設定 (初回必須)
画面右上の **設定(⚙️)** ボタンをクリックして以下を設定します。

1. **GitHub Personal Access Token (PAT)**:
   - [GitHub Settings](https://github.com/settings/tokens) で `repo` スコープを許可したトークンを発行。
   - `ghp_...` で始まる文字列を入力して保存します。
2. **GitHub Owner / Repository**:
   - リポジトリをフォークした場合は、ご自身のユーザー名とリポジトリ名を入力してください。
   - ※ URLから自動取得可能な場合は、空欄のままでも動作します。

## 3. 機能解説

### 3.1. Journal (日報)
- **リッチエディタ**: Markdown形式で入力。ツールバーから太字、リスト、リンク、画像などを挿入可能。
- **オートセーブ**: 入力内容は5秒ごとにブラウザの `localStorage` に保存され、ページを閉じても復元されます。
- **PDF印刷**: 「🖨️ 印刷」ボタンをクリックすると、UIを除外した状態で綺麗にPDF出力されます。

### 3.2. Tasks (タスク管理)
- **新規タスク**: 「+ 新規」ボタンから追加。カテゴリ、優先度、期限を設定します。
- **検索・フィルタ**: 検索バーでタイトルや詳細を瞬時に絞り込めます。
- **ソート**: 期限が近い順や、作成日が新しい順などに並べ替え可能です。

---

## 4. デスクトップ版アプリ (Python)
複雑な集計（Dashboard, Monthly Report）やローカルバックアップを行いたい場合に使用するサブツールです。

### 起動方法
1. `Run_LogManager.bat` を実行。
2. 起動時にGitHubから最新データを Pull し、終了時に Push します。

---

## 5. トラブルシューティング
- **同期エラー**: PATの期限切れや、リポジトリ名の設定ミスがないか確認してください。
- **文字化け**: 全てのデータファイル（JSON）は UTF-8 で保存されます。
