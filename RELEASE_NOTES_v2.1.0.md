# MP-LogManager v2.1.0 (GAS Edition) Release Notes

## 🚀 概要
モバイルでの実運用における安定性とユーザーエクスペリエンスの向上を目的としたブラッシュアップ・アップデートです。

### ✨ 新機能・改善点
- **通信の安定化 (Exponential Backoff)**:
  - 通信エラー時に最大3回までの指数バックオフを伴う自動リトライを実装。
  - リクエストのタイムアウト処理（10秒）を導入し、不安定なネットワーク環境でのフリーズを防止。
- **接続テスト機能**:
  - 設定画面に「接続テスト」ボタンを追加。GAS URLが正しく設定されているかその場で確認可能に。
- **モバイル UX の最適化**:
  - iOS (Safari) での入力フォームフォーカス時の意図しないズーム動作を抑制 (font-size 16px 以上を担保)。
  - iPhone 等の SafeArea を考慮したレイアウト調整。
  - タップ領域の拡大 (44px) により、操作ミスを軽減。
- **トースト通知の導入**:
  - `alert()` による画面のブロッキングを廃止し、非同期のトースト（スナックバー）通知へ移行。

### 🔧 修正・調整
- 既存のすべての `alert()` 呼び出しを `showToast()` に置き換え。
- `openspec` docs の内容を Web App 専用リポジトリの状態に合わせた最終メンテナンス。
