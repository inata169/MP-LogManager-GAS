# アイコン画像について

このディレクトリには以下のアイコン画像を配置してください:

- `icon-192.png` (192x192px)
- `icon-512.png` (512x512px)

## デザイン推奨仕様
- 背景色: #1a1a2e (ダークネイビー)
- シンボル: 白いチェックマーク + 医療十字マーク
- スタイル: ミニマル、プロフェッショナル

## 簡易作成方法
オンラインツール（例: Canva, Figma）で以下の手順で作成できます:
1. 512x512pxのキャンバスを作成
2. 背景を #1a1a2e で塗りつぶし
3. 白い✓マークと＋マークを中央に配置
4. PNG形式でエクスポート
5. 192x192pxにリサイズしたバージョンも作成

または、既存のフリーアイコンサイト（例: flaticon.com）から
医療系のチェックリストアイコンをダウンロードして使用することもできます。
